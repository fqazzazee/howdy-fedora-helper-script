## Getting Help

1. Run the diagnostic: `sudo ./install-howdy.sh --diagnose`
2. Check the [[FAQ]] for common issues
3. Review the troubleshooting section in [HOWDY-MANUAL](../blob/main/HOWDY-MANUAL.md)
4. Open an issue with diagnostic output if the problem persists
